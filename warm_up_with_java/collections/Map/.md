# Map idea (very simple)
`Map<K, V>` stores key -> value pairs.

- key = unique (no duplicate keys)
- value = can be duplicate
- example: roll -> studentName, email -> phone, productId -> price

- Syntax:
`Map<String, Integer> map = new HashMap<>();`

## Most important Map methods
- `put(key, value)` -> add/update
- `get(key)` -> get value
- `containsKey(key)` -> key exists?
- `containsValue(value)` -> value exists?
- `remove(key)` -> delete by key
- `size()`, `isEmpty()`
- `entrySet()` -> iterate key+value
----
### HashMap vs LinkedHashMap vs TreeMap
+ HashMap -> fastest, no order guarantee
+ LinkedHashMap -> insertion order maintained
+ TreeMap -> sorted by key
